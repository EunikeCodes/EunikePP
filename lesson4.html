HTML Elements

This table includes basic HTML elements and what they are used for



HTML Elements What is it for
< h1 > This tag is used to create the first/biggest heading
< p > This is used to create paragraphs
< body > Used to create the body area for the website
< head > Similar to the body element, this is used to create a head area for the website. This is placed before the body tags.
< br > This is used to create space, it is basically like enter in the keyboard
< nav > This element is used to create the navigation system
< !DOCTYPE html > This element is what makes the document HTML. It must always be included when making a website.
< mark > Used to highlight text.
< em > Used to italicize text.
