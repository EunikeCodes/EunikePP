Link

In HTML, you can add links to sites, videos, ...



Codes Used:

  1. First, you need a < a > tag. The code is:

    < a href="the url/link" >Description of where it would link to< /a >

    (Make sure there is no space between the tags)
  2. This code could also be used in navigation systems when wanting to link to another page in the website
