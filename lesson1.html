Adding Pictures

There are specific codes needed in order to add pictures into the website. In order to do so, HTML will be used.



Codes Used:

  1. img src="imagename.png/jpg or the url" (Make sure to put the text in between these tags <>)
  2. You could put an alt atribute, which will be a caption showed if the viewer for some reason is unable to view the picture.
    The code is:

    alt="description of picture"

    This should be placed after the first code.
    An example of the first two combined is:

    img src="url,..." alt="description"

    (Do not forget to put the tag < before the word img and > after the "description")
  3. To change the size of the image you will need the width and height attributes.
    The code is:

    width="__px" height="__px"

    An example of the whole code combined is:

    img src="url,..." alt="description" width="__px" height="__px"

    (Do not forget to put the tag < before the word img and > after the last "__px")
