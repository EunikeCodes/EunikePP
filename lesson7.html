CSS Part 2

The style/design for the website



Fonts:

  1. There are many types of codes used that can change the Font. The first one is font-style, which can allow the text to be italicized or normal. An example if you want to change the paragraph font is:

    p {
    font-style:italic;
    }
  2. You could also change the font size. Continuing from above the code is:

    p {
    font-style:italic;
    font-size:2px;
    }
  3. There is also font family, which is like Times New Roman, Arial, ... The code used to change this is:

    p {
    font-style:italic;
    font-size:2px;
    font-family:Arial;
    }
