Lists (Bulletpoints)

There are two types of list. One is called unordered list (ul), which is basically bulletpoints. The other one is called ordered list, which is when numbers are used (ol)



Codes Used for Unordered List:

  1. A ul tag is needed.
    The code is:

    < ul >
    (Do not add space between the bracket tag and ul)
  2. After the ul tag, a li tag is needed. The li tag symbolizes every new bulletpoint.
    The code is:

    < li >Words...< /li >
    (Do not put space between the tag and li *the one showed above had space in them. For this, there is a thing called opening and closing tags. The opening tag does not require /, while the closing tag does as shown above)

    An Example of the first two combined is:

    < ul >
    < li >words...< /li >
    < li >words...< /li >
    < li >words...< /li >

    (You can put as much li as you want. It depends on how many bulletpoints you want. *Do not put space in between the tag <> and the attributes when actually writing the codes.)
  3. Lastly, make sure to add the closing tag for the ul attribute
    The code is:

    < /ul >

    (Again do not put space)
    An example of all three combined is:

    < ul >
    < li >words...< /li >
    < li >words...< /li >
    < li >words...< /li >
    < /ul >
