Lists (Numbered Bulletpoints)

There are two types of list. One is called unordered list (ul), which is basically bulletpoints. The other one is called ordered list, which is when numbers are used (ol)



Codes Used for Ordered List:

  1. An ol attribute is needed.
    The code is:

    < ol >
    (Do not add space between the bracket tag and ol)
  2. After the ol attribute, a li attribute is needed, similar to the unordered list.
    The code is:

    < li >words...< /li >
    (Exactly the same as the unordered list. The number of li depends on how many bulletpoints you want. Do not put space between the tag and attribute)
    An example of the first two combined is:

    < ol >
    < li >words...< /li >
    < li >words...< /li >
    < li >words...< /li >
    (You can put as many li as you want)
  3. Lastly, make sure to put the closing tag.
    The code is:

    < /ol >
    (Make sure there is no space for all the attributes and tags)
    An example of everything combined is:

    < ol >
    < li >words...< /li >
    < li >words...< /li >
    < li >words...< /li >
    < /ol >
