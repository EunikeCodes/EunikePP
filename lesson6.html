CSS Part 1

The style/design for the website



Introduction:

  1. The first thing to do is to create a stylesheet. In order to do so, open a new page in atom and save as "stylesheet.css"
  2. The second thing that needs to be done is to link the html to css. In each of the html pages, you must add the following code in the heading:

    < link type="text/css" rel="stylesheet" href="stylesheet.css" media="screen" >

  3. The stylesheet does not require a heading or body. Leave it blank until you want to add codes that can change the appearance of the website.

Colors:

  1. For CSS, colors are coded through HEX value or the name of the color itself. In order to change the background color of the body area to black for example, then use the following CSS code (Put it into the stylesheet):

    body {
    background-color:black;
    }

    (Instead of using the <> tags, use {} for CSS)
  2. If you want to change the text color in a specific area, for example the body area then just add this code:

    body {
    background-color:black;
    color:white;
    }
